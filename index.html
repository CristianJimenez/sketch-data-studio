Sketch Financial Data Generator

A Sketch plugin to automatically generate data tables and charts

View on GitHub Download .zip Download .tar.gz

Generating tables and charts just got a lot easier!

Get Started Now

  1. Download the .zip or clone the Github repo above.
  2. In the Sketch App, click "Plugins > Reveal Plugins Folder" from the top menu.
  3. Place the "sketchfindatagen" folder in your plugins folder
  4. Generate some charts and tables.

Authors and Contributors
